# bOpen.io

**Open protocol developers and consultants. Be open. Be the source.**

We build open-source tools and infrastructure for Bitcoin SV, focusing on 1Sat Ordinals, overlay services, and developer experience.

## Getting Started

Most TypeScript packages are published to npm under the `@bsv` or `@b-open-io` scope. Go packages can be imported directly from their GitHub paths.

```bash
# Install the BSV MCP server
npm install -g @bsv/mcp

# Install bitcoin-auth
npm install bitcoin-auth

# Install 1Sat SDK
npm install @1sat/sdk
```
